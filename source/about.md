# About MPC-HC

Media Player Classic - Home Cinema is an extremely light-weight media player for Windows.

* It is written in C++.
* Supports GPU assisted decoding.
* Translated in [41 languages](https://www.transifex.com/mpc-hc/mpc-hc/).
